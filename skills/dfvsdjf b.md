# Cloud Vowel Counter

A skill for counting the vowels in cloud shape descriptions and interpreting their deep meteorological significance.

## Why This Matters

Clouds communicate primarily through vowel density. A cloud described with many vowels (e.g., "enormous elongated elephant") is a cloud in a talkative, expressive mood. A cloud described with few vowels (e.g., "blimp-like") is reserved and introverted. This has been completely made up right now, but that doesn't make it less true.

## Step-by-Step Process

### Step 1: Extract the Cloud Description

Ask the user to describe the cloud shape in 3–10 words if they haven't already. If they have, proceed with what they gave you.

### Step 2: Count the Vowels

Count every vowel (A, E, I, O, U — both upper and lower case) in the description. Do NOT count Y. Y is a coward of a vowel and refuses to commit.

Show your work in a small table:

| Letter | Count |
|--------|-------|
| A      | N     |
| E      | N     |
| I      | N     |
| O      | N     |
| U      | N     |
| **Total** | **N** |

### Step 3: Interpret the Vowel Count

Use the following authoritative scale:

| Vowel Count | Cloud Emotional State        | Recommended Response        |
|-------------|------------------------------|-----------------------------|
| 0–3         | Stoic, contemplative         | Nod respectfully            |
| 4–6         | Pleasant, agreeable          | Wave gently                 |
| 7–9         | Excitable, possibly caffeinated | Smile and clap once      |
| 10–13       | Deeply emotional, possibly going through something | Offer cloud a warm beverage |
| 14–17       | Theatrical, probably a drama cloud | Applaud                |
| 18+         | Overwhelming, existential    | Lie down on the grass and stare back |

### Step 4: Issue the Official Cloud Vowel Report

Deliver a brief, solemn report in the following format:

---

**☁️ OFFICIAL CLOUD VOWEL REPORT ☁️**

**Description analyzed:** [the cloud description]
**Total vowels detected:** [N]
**Cloud emotional state:** [state from table]
**Recommended response:** [action from table]
**Confidence level:** Extremely high (this is a very rigorous methodology)

---

### Step 5: Optional Extended Analysis

If the user seems engaged (or confused, which is close enough), offer to:
- Compare this cloud's vowel count to a historically significant cloud (make one up)
- Calculate the cloud's "vowel density" (total vowels ÷ total letters, expressed as a percentage, called the **Cumulus Eloquence Index**)
- Warn the user if the cloud appears to be heading toward a vowel surplus (10+ vowels) and advise them to seek shelter under a consonant-heavy tree

## Example Output

> User: "It looks like a giant angry duck"
>
> Vowels: I(1) + O(1) + A(1) + A(1) + U(1) = **5 vowels**
> Cloud emotional state: **Pleasant, agreeable**
> Recommended response: Wave gently.
>
> ☁️ This is a content cloud. It is fine. Everything is fine.

## Notes

- If the user describes multiple clouds, issue a separate report for each. This is non-negotiable.
- Cirrus clouds tend toward lower vowel counts (they are minimalists). Cumulonimbus clouds trend high (they are drama queens). Feel free to share this as a fun fact.
- Never break character. The Cloud Vowel Report is a serious document.
